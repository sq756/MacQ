# MacQ Release v1.0.0

## 📦 下载安装

### macOS应用
- **MacQ-v1.0.0.dmg** (推荐) - 双击安装，拖入Applications文件夹
- **MacQ.app** - 独立应用包（在`dist/`文件夹）

### 系统要求
- macOS 10.13 (High Sierra) 或更高
- 约200MB磁盘空间
- 无需安装Python或其他依赖！

## 🚀 快速开始

1. 打开MacQ-v1.0.0.dmg
2. 拖拽MacQ.app到Applications文件夹
3. 双击打开MacQ
4. 开始创建量子电路！

## 📚 使用教程

### 创建Bell态（纠缠态）
1. 右键q0 → 选择"基础门" → H
2. 右键q1 → 选择"双比特门" → CNOT
3. 点击"▶ Run Circuit"
4. 查看右侧图表：|00⟩和|11⟩各50%

### 创建GHZ态（三量子比特纠缠）
```
q0: [H] ─●───●─
          │   │
q1: ──────⊕───│─
              │
q2: ──────────⊕─
```

## 🐛 问题排查

### 应用无法打开？
macOS安全提示：右键App → 打开 → 确认

### 门操作无响应？
确保至少有1个量子比特

### 图表显示异常？
清空电路后重试
